- NPMS:

  - Dependencies:

    - cors
    - uuid
    - express
    - bcryptjs
    - dotenv
    - joi
    - jsonwebtoken

  - Dev dependencies:

- How To Install

  - npm init -y
  - npm i cors uuid express bcryptjs dotenv joi jsonwebtoken
  - npm i --save-dev nodemon
